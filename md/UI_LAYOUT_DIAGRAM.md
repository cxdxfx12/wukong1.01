## 规则说明对话框

```
┌─────────────────────────────────────────────────────────────────┐
│  运费计算规则说明                                           ✕  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  # 运费计算规则说明                                              │
│                                                                  │
│  📋 目录                                                         │
│  1. 计算模式说明                                                 │
│  2. 重量段划分                                                   │
│  3. 区域划分                                                     │
│  4. 计算公式                                                     │
│  5. 特殊规则                                                     │
│  6. 常见问题                                                     │
│                                                                  │
│  ─────────────────────────────────────────────────────────────  │
│                                                                  │
│  ## 计算模式说明                                                 │
│                                                                  │
│  系统支持 4 种计算模式：                                          │
│                                                                  │
│  ### 1. 实际重量模式                                             │
│  - 定义：使用实际重量和体积重量的最大值                          │
│  - 公式：有效重量 = max(实际重量，体积重量)                      │
│  - 适用：大多数普通客户                                          │
│                                                                  │
│  ### 2. 拉均重模式                                               │
│  - 定义：在有效重量基础上，根据平均重量参数加价                  │
│  - 加价规则：                                                    │
│    • 基准重量：0.5kg（默认）                                     │
│    • 上限重量：1.0kg（默认）                                     │
│    • 递增步长：0.1kg（默认）                                     │
│    • 每步加价：0.1 元（默认）                                      │
│                                                                  │
│  ### 3. 百克续模式                                               │
│  - 定义：续重按 100 克为单位计算（向上取整）                        │
│  - 公式：运费 = 首重价 + ceil(续重重量 × 10) × (续重价 / 10)     │
│                                                                  │
│  ### 4. 全续模式                                                 │
│  - 定义：重量向上取整后计算续重                                  │
│  - 公式：运费 = 首重价 + ceil(续重重量) × 续重价                 │
│                                                                  │
│  ─────────────────────────────────────────────────────────────  │
│                                                                  │
│  ## 重量段划分                                                   │
│                                                                  │
│  ### 一区（江苏/浙江/安徽/上海）                                 │
│                                                                  │
│  | 重量段 (kg) | 首重价 (元) | 续重价 (元/kg) | 说明     |      │
│  |------------|-----------|--------------|----------|            │
│  | 0 - 0.5    | 2.26      | 0            | 固定价格 |            │
│  | 0.5 - 1    | 2.46      | 0            | 固定价格 |            │
│  | 1 - 2      | 3.56      | 0            | 固定价格 |            │
│  | 2 - 3      | 4.76      | 0            | 固定价格 |            │
│  | 3 - 30     | 4.76      | 0.8          | 首重 + 续重 |         │
│  | 30+        | 3.86      | 0.8          | 大货价格 |            │
│                                                                  │
│  [更多内容...滚动查看]                                           │
│                                                                  │
│  ─────────────────────────────────────────────────────────────  │
│                                                                  │
│  ## 常见问题                                                     │
│                                                                  │
│  ### Q1: 为什么 3.0kg 和 3.1kg 的运费相差不大？                   │
│  A: 3kg 是价格段分界点。3kg 以内是固定价格，超过 3kg 后按"首重 + 续重" │
│     计算。系统已优化价格段衔接，确保运费连续增长。               │
│                                                                  │
│  ### Q2: 体积重如何计算？                                        │
│  A: 体积重 = 长 × 宽 × 高 / 抛比系数（默认 6000 或 8000）          │
│     系统会取实际重量和体积重量的最大值作为计费重量。             │
```

## 按钮功能对比

| 按钮 | 功能 | 目标用户 |
|------|------|---------|
| **规则管理** | 管理客户专属价格规则 | 管理员/操作员 |
| **全局规则** | 设置活动加价、临时加价等全局参数 | 管理员 |
| **规则说明** (新增) | 查看计算规则、公式、常见问题说明 | 所有用户（特别是新手） |

## 使用场景

### 场景 1：新员工培训
- 新员工不熟悉计算规则
- 点击"规则说明"按钮查看完整的计算逻辑
- 快速了解 4 种计算模式的区别

### 场景 2：客户咨询
- 客户询问"为什么 3.1kg 的运费比 3.0kg 贵？"
- 打开"规则说明"，查看"重量段划分"章节
- 用实际案例解释给客户听

### 场景 3：问题排查
- 发现运费计算结果异常
- 查看"计算公式"章节，确认计算方法
- 对照"常见问题"排查可能的问题
